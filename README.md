# 💬 WebSocket Chat App

A real-time chat application built using WebSocket, React, and Node.js. This project allows multiple users to communicate instantly with live updates and typing indicators.

---

## 🚀 Features

* 🔴 Real-time messaging
* ✍️ Typing indicator
* 👥 Multiple users support
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Backend:** Node.js
* **Communication:** WebSocket

---

## 📁 Project Structure

```
websocket-chat/
│── backend/      # Node.js server
│── frontend/     # React app
```

---

## ▶️ Run Locally

### 1. Clone the repository

```
git clone https://github.com/Rohan123raj/websocket-chat.git
```

### 2. Backend setup

```
cd backend
npm install
node server.js
```

### 3. Frontend setup

```
cd frontend
npm install
npm run dev
```

---

## 🌐 Access the App

Open your browser and go to:

```
http://localhost:5173
```
